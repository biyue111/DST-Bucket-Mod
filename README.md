# DST-Bucket-Mod
Bucket mod for Don't Starve Together

**Name**: Bucket

**Crafting**:
    Board * 2,
    Rope * 1

**Tab**:
    Survival

**Tier**:
    Always available

**Durability**:
    20 uses (get water)

**Perk**:
    Get water from pools;
    Keep water;
    Quenach;
    Make ice

**Stacks up to**:
    Does not stack
    
**Debug Spawn**:
    "bucket"

**Variable**: 

    State (min = 0%, max = 100%, by = 25%)
    Durability (min = 0%, max = 100%, by = 5%)
