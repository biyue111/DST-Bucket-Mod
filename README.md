# DST-PigTown-Mod
PigTown mod for Don't Starve Together

## Basic Information

**Name**: PigTown

**Description**: A town including creatures extended from "pigman"

**Tab**: Creature(s)

## List:
* Normal Pigman
* Soldier Pigman
* Merchant Pigman
